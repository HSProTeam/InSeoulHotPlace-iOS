# In Seoul Hot Place 
서울 혼잡도를 알려주는 서비스 🌝

</br>

### 🏷 Commit Message
- `[Add]` 기능 외에 추가사항 (파일, 프로토콜, 익스텐션 등)  
- `[Del]` 파일, 에셋 삭제
- `[Feat]` 새로운 기능 추가
- `[Fix]` 에러 및 버그 수정, 기능에 대한 수정 작업
- `[Edit]` Fix 외에 모든 수정 작업 (파일 위치, 주석, 코드 스타일 등)
- `[Docs]` 리드미 및 문서 수정
- `[Style]` UI 관련 작업
